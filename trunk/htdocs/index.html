Pyxine is a set of Python bindings and an object-oriented wrapper for xine.

Using Pyxine it is possible to write simple (or complex) movie players in the Python programming language.

First release! Get it here.

This code is in it's very infancy, but it seems to work more-or-less --- at least for me. works for me. Expect much refactoring and numerous API changes within the near future.

There's a pyxine-devel mailing list, which you should feel welcome to join if you're interested in the development of this code.
